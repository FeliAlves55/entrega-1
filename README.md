INSTALACION DE LOS ENOTRNOS
Esta es una breve descripcion de como descargar los entornos para la ejecucion del programa:

 - Python 3.13.12
 
 - Jupyter Notebook
Primer Paso:
Descargar python en su pagina original: https://www.python.org/ (Verificar la version a la hora de instalarla)

Segundo paso
Descargar el jupyter, pegando este comando en la consola:

pip install notebook

  
Ejecución del Programa

1- Estos archivos estan dentro de una carpeta en el repositorio remoto (src/notebooks)

2- Una vez descargado el programa, abrir el archivo con VSC (Visual Studio Core, u otro IDE)

3- Dentro les dire que descargue las extensiones, Python Microsoft y Jupyter Microsoft. En la barra de herramientas a la izquierda, una con muchos cuadrados esa son las extensiones.

4- Finalmente abra el archivo, con .ipynb, seleccionar el interprete de Python y luego ejecute el programa o la celda que desee
